# 📚 책뜰 (BookGarden)

책뜰(BookGarden)은 사용자가 읽은 책을 저장하고 관리할 수 있는 웹 애플리케이션입니다. '나만의 온라인 책꽂이'를 만들어 책을 추가하고, 수정하고, 삭제하며 나만의 도서 목록을 관리할 수 있습니다.

---

### **🛠️ 기술 스택**
- **Frontend:**
  - Next.js (pages router)
  - Tailwind CSS (스타일링)
  - TypeScript
  - React Query (데이터 관리)
    
- **Backend:**
  - Supabase (서버리스 데이터베이스, 인증, 파일 저장소)
